# Data Ethics and Privacy

This project is intended for personal, non-commercial aggregate analysis of public Reddit
community overlap among Warhammer 40k faction communities.

Principles:

- Do not publish usernames.
- Do not publish raw comments or submissions.
- Do not publish individual user histories.
- Store and share only aggregate subreddit-level statistics.
- Keep Reddit credentials local in `.env`.
- Respect Reddit API terms, rate limits, and community norms.
- Use small samples by default and document sampling limitations.
