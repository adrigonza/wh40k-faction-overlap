What this project measures

The project estimates subreddit overlap using public activity and turns it into aggregate network graphs.

Overlap probability

Estimate how often users sampled from a faction community also appear in another public subreddit.

Affinity / lift

Normalize against baseline popularity so huge subreddits do not dominate the results.

Faction similarity

Compare faction communities based on their full public subreddit participation profiles.

Privacy-first design

This project is designed to produce community-level aggregate statistics only. It does not post, comment, vote, message users, moderate communities, or publish individual user histories.

Planned stack

Python PRAW pandas NetworkX matplotlib scikit-learn CSV exports

Status

Early planning and prototype stage. The first milestone is a read-only local script that exports anonymized aggregate CSV files and simple network graphs.
